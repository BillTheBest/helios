  • JMXMetric modes:
    • One2One: One MBean per metric. The ObjectName of the MBean has 1:1 cardinality with the metric FQN
    • NameSpace: The ObjectName and MBean represents the lowest (most specific) portion of the metric namespace. The metric name is an attribute of the MBean and the value of the attribute is the OpenTypeMetric CompositeType.
    • NameSpaceTable: Same as NameSpace except the metrics are represented as one attribute in a TabularType.
